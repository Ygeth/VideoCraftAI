# **App Name**: VideoCraft AI

## Core Features:

- Content Generation: Generate video scripts based on user input.
- Script Editor: Allow users to edit and refine generated video scripts.
- Text to video tool: Converts the video script into video content using AI powered algorithms, images, transitions and music.
- Testing Page: Dedicated space for developers to experiment with the system.
- Final Preview: Before render, give user a clear view of all media. Allowing last minute adjustment. Media, subtitles and music. A tool decides to use those elements.
- Video Rendering: Processes edited scripts into complete video files.
- User account creation: Allows user creation using only their email

## Style Guidelines:

- Primary color: Dark forest green (#306754) to evoke growth, learning, and video creation.
- Background color: Very light green (#F0FAF5) to create a calm, productive, neutral space.
- Accent color: Burnt orange (#C06014) for a unique contrast.
- Body and headline font: 'Alegreya', a humanist serif.
- Simple, line-based icons representing video creation and editing tools.
- Clean and organized layout for script editing and previewing, prioritizing user focus.
- Subtle animations on content generation and preview loading.
